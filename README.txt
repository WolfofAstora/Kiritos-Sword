KiritosSword Mod
================

Adds the KiritosSword item to Minecraft using Forge.

How to build:
-------------
1. Make sure you have Java 21 installed.
2. Run `./gradlew build` in this directory.
3. The mod jar will be in `build/libs/`.

How to use:
-----------
- Place the jar in your Minecraft mods folder.
- The sword will be available in-game.
